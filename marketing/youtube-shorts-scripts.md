# 유튜브 쇼츠 스크립트 (30초~60초)

## 쇼츠 1: "내 사주가 음악이 된다고?"

**길이**: 45초
**형식**: 화면녹화 + 자막 + BGM (사주해 앱 사주음악 중 하나)

```
[0-5초] 검은 화면 + 텍스트
"사주앱인데 음악 만들어주는 거 처음 봄"

[5-12초] sajuhae.vercel.app 접속 화면
자막: "사주해라는 앱인데..."
       "생년월일 넣으면 AI가 사주 분석해줌"

[12-20초] 생년월일 입력 → 결과 화면
자막: "오행 분포, 십신, 용신까지 다 나옴"

[20-30초] 사주 음악 페이지 이동 → 재생
자막: "근데 진짜 신기한 건..."
       "내 오행으로 음악을 만들어줌!!"
(실제 사주 음악 BGM이 흘러나옴)

[30-38초] 장르 변경 (국악 → 로파이 → 오케스트라)
자막: "장르도 7개나 됨ㅋㅋ"
       "국악, 피아노, 로파이, 오케스트라..."

[38-45초] 엔딩 카드
텍스트: "당신의 사주가 음악이 되고, 영상이 됩니다."
        "무료 → sajuhae.vercel.app"
        "#사주 #AI사주 #사주음악 #사주해"
```

## 촬영 가이드

### 필요 장비
- 스마트폰 화면 녹화 (또는 PC 화면 녹화)
- 자막 편집 앱 (CapCut, VLLO 등)

### 촬영 팁
1. **세로 9:16 비율** (1080x1920)
2. **첫 3초가 핵심** — 강렬한 후킹 텍스트
3. **BGM** — 사주해 앱의 실제 사주 음악 사용 (저작권 문제 없음, 자체 생성 음악)
4. **자막** — 큰 글씨, 밝은 색, 검은 테두리
5. **속도감** — 빠른 컷 전환 (2-3초 단위)
