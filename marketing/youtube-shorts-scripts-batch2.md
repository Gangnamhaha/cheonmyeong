# 유튜브 쇼츠 스크립트 2차 (30초~45초)

## 쇼츠 4: "시대극 무드 쇼케이스"

**길이**: 35초  
**형식**: 화면녹화 + 자막

```
[0-4초] 후킹
텍스트: "사주앱인데 시대극 무드 영상까지 된다고?"

[4-10초] 메인 화면 스크롤 → 쇼케이스 진입
자막: "사주해 메인에서 바로 확인 가능"

[10-18초] '🏯 시대극' 버튼 클릭
자막: "장르별 무드 선택 가능"

[18-28초] 쇼케이스 프리뷰 재생
자막: "역사 드라마 느낌의 운명 연출"

[28-35초] 엔딩
텍스트: "당신의 사주가 음악이 되고, 영상이 됩니다"
       "무료 체험: sajuhae.vercel.app"
```

## 쇼츠 5: "리포트 기본 vs 프로 10초 정리"

**길이**: 40초  
**형식**: 텍스트 카드 + 화면녹화

```
[0-5초] 후킹
텍스트: "사주 리포트, 기본이랑 프로 뭐가 다를까?"

[5-14초] 기본 플랜 화면
자막: "기본(₩9,900)"
      "7개 카테고리 심층 분석"

[14-24초] 프로 플랜 화면
자막: "프로(₩25,000)"
      "대운 10년 + 세운 3년 + 인생 조언"

[24-33초] 비교 카드
자막: "핵심 정리 = 기본"
      "장기 플랜 = 프로"

[33-40초] 엔딩
텍스트: "사주해 프리미엄 리포트"
       "sajuhae.vercel.app/pricing"
```

## 쇼츠 6: "궁합 리포트 저장 활용법"

**길이**: 45초  
**형식**: 나레이션형 + 자막

```
[0-5초] 후킹
텍스트: "궁합은 보고 끝내면 아까움"

[5-12초] 궁합 입력 화면
자막: "두 사람 생년월일 입력"

[12-22초] 궁합 결과 화면
자막: "성격/연애/직장 궁합까지 분석"

[22-34초] 리포트 다운로드 화면
자막: "문서(DOCX)로 저장해서"
      "나중에 다시 보기"

[34-45초] 엔딩
텍스트: "연애·결혼·동업 고민할 때"
       "정리본이 있으면 대화가 쉬워짐"
       "sajuhae.vercel.app/gunghap"
```
