# 크리에이터/제휴 아웃리치 트래커

## 사용법
- 1행 = 1명(또는 1개 커뮤니티)
- 상태는 `미접촉 → 발송 → 회신 → 협의중 → 진행중 → 완료`로 관리

| 날짜 | 채널 | 대상명 | 팔로워/규모 | 담당자 | 제안유형 | 발송문구버전 | 상태 | 회신일 | 단가/조건 | 진행링크 | 비고 |
|---|---|---|---:|---|---|---|---|---|---|---|---|
| 2026-03-23 | 인스타 | 예시크리에이터A | 32,000 | 민수 | 쇼츠 1건 | DM-v1 | 발송 |  |  |  |  |
| 2026-03-23 | 네이버카페 | 예시카페B | 18,000 | 민수 | 체험 이벤트 | Cafe-v1 | 회신 | 2026-03-24 | 공지 1회 |  |  |
